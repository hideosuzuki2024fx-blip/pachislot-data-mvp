# 🎯 【黒字化検証フェーズ】パチスロ収支・期待値特化型PWA MVP

## 1. プロジェクト概要
本プロジェクトは、パチスロ愛好家をターゲットとし、**黒字化**を目的としたニッチ特化型サブスクリプションWebアプリ（PWA）のMVP（最小実行可能製品）開発プロジェクトです。

**最大の目標は、2週間の市場検証を通じて「売れる証拠」を取得することです。**

## 2. コア・バリュー（ニッチ・ブルーオーシャン要素）
既存アプリが抱える「手動記録の煩雑さ（P1）」と「分析の難しさ（P2）」を解消するため、以下の2つの核となる機能に極限まで集中します。

| ID | 機能名 | 解決するPain Point |
| :--- | :--- | :--- |
| **P1** | **ゼロ手間記録** | 記録作業を10秒以内に簡略化し、記録漏れを解消します。 |
| **P2** | **行動変容期待値レポート** | 記録データに基づき、ユーザーの勝率向上に直結する期待値傾向を自動可視化します。 |

## 3. 技術スタック
* **PWA/Frontend:** Next.js (React)
* **Styling:** Tailwind CSS
* **Backend/Auth/DB:** Supabase (MVPフェーズで導入)

## 4. プロジェクト状態
プロジェクトの状態と決定事項は `docs/state/phase.yml` が唯一の真実です。

---
### 👨‍💻 開発環境のセットアップ (ローカルでの実行手順)

以下のコマンドをターミナルで実行してください。

1. **依存関係のインストール:**
   `npm install`
2. **ローカル開発サーバーの起動:**
   `npm run dev` 
   (http://localhost:3000 でアクセス可能になります)
